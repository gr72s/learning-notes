# 源码分析
## 文件结构(src)
* additional-components, 附加组件: 提供额外功能, 比如画布上的略缩框、放大缩小控制按钮等
* components, 组件
* container, 图元容器
  * EdgeRenderer, 边缘渲染器
  * FlowRenderer, 流渲染器
  * GraphView, 画布
  * NodeRenderer, 节点渲染器
  * ReactFlow
  * ZoomPane, 变焦窗口
* contexts: 定义react context
* hooks: 定义react hooks
* store: 定义redux store
* types: 定义redux action
* utils: crud元素的工具类
